## Screenshot Guidelines

- Use high-resolution screenshots (at least 1920x1080)
- Ensure any sensitive information (tokens, credentials) is redacted
- Capture the full browser window or relevant UI section
- Use consistent browser and zoom level across all screenshots
- Consider adding annotations or highlights to key UI elements
- Save as PNG format for best quality
